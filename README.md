# Nautilus Trader Development Skills

Skills for building with and building on NautilusTrader.

## Setup

For best results, clone NautilusTrader locally:

```bash
git clone https://github.com/nautechsystems/nautilus_trader.git
```

When a domain skill activates, it will ask for your local NT path.
If you don't have a local clone, it falls back to browsing the official
repo on GitHub.

## Skills

### Phase Skills (Using NT)

| Skill | Purpose |
|-------|---------|
| nt-architect | Translate research → NT component architecture |
| nt-implement | Templates and examples for building NT components |
| nt-review | Validate implementations before deployment |

### Domain Skills (Building NT)

| Skill | Purpose |
|-------|---------|
| nt-core-dev | Data model, message bus, execution pipeline, FFI |
| nt-adapter-dev | Building exchange/data adapters |
| nt-strategy-dev | Building Actors/Strategies in Python and Rust |
| nt-analysis-dev | Statistics, tearsheets, backtest reporting |
| nt-persistence-dev | Catalog, data lake, serialization |
| nt-kernel-dev | Kernel/Engine/Controller, boot sequence |
| nt-backtest-dev | SimulatedExchange, fill models, data replay |

### Learning Pathway

| Skill | Purpose |
|-------|---------|
| nt-learn | Structured curriculum from beginner to NT dev |
